# Hawa Studios Reference Profile

This profile contains reference images and style definitions for Hawa Studios, an Afghan music label with a distinctive visual identity.

## Brand Identity

- **Header:** "HAWA STUDIOS PRESENTS" at the top of all covers
- **Typography:** Clean, bold sans-serif for titles, mixed with script elements for specific styles
- **Photography:** Professional portrait photography with high production value
- **Color Treatment:** Distinctive color grading specific to each style category

## Style Categories

1. **Wedding/Celebration**
   - Elegant floral backgrounds
   - Warm gold gradients
   - Formal portrait composition
   - Example: Omar Sharif "LIVE WEDDING MIX"

2. **Modern/Contemporary**
   - Dynamic compositions
   - Bold color accents or monochromatic with single color highlight
   - Modern typography treatments
   - Examples: Tawab Jawed "Herati REMIX 2026", Duran Shewaki "Majlisi"

3. **Traditional Portrait**
   - Formal attire
   - Classic portrait composition
   - Clean typography
   - Examples: Nazir Khara "Tor e Sterge", Nazir Khara "Paiwand e Ishq"

4. **Performance/Entertainment**
   - Stage/venue settings
   - Theatrical lighting
   - Performance-focused imagery
   - Example: Waheed Soroor "Dukhtar Herati"

## Usage Guidelines

When generating covers for Hawa Studios:

1. Always include "HAWA STUDIOS PRESENTS" header at the top
2. Maintain the typography hierarchy appropriate to the selected style
3. Use professional portrait photography as the primary visual element
4. Apply color grading consistent with the chosen style category
5. Ensure bilingual text is properly balanced when applicable

## Reference JSON Files

Each style category has its own JSON definition file with detailed specifications for:
- Visual elements
- Typography treatments
- Color palettes
- Mood and compositional guidelines

These can be used to generate style-specific prompts for the cover art generator.
